# Información
El concepto de Mario, el objetivo del juego es derrotar a todos los enemigos en cada fase

# Controles
<b>W:</b> Saltar
<b>A:</b> Muévete hacia la izquierda
<b>S:</b> Agáchate
<b>D:</b> Muévete hacia la derecha
<b>Shift / Ctrl:</b> Correr
<b>M:</b> Silenciar
